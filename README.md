🌾 Crop Yield Prediction & Pest Monitoring System

A Machine Learning–based web application that helps users predict crop yield, monitor pest conditions, and access real-time weather forecasting. The project is designed to support smart agriculture and improve farming decisions using ML models and web technologies.

🚀 Features
🌾 Crop Yield Prediction using Random Forest Regression
🐛 Pest Monitoring using Gradient Boosting
🌦️ Real-time Weather Forecasting using Weather API
🌍 Weather support for 80+ cities
📊 Predicts yield for 20–25 crops
🌐 User-friendly web interface
⚡ Flask-based backend integration
🛠️ Technologies Used
1.Frontend-HTML,CSS,JavaScript,Bootstrap
2.Backend-Python,Flask
3.Machine Learning-Random Forest Regression,Gradient Boosting,Scikit-learn-Pandas,NumPy
4.API-OpenWeatherMap API
